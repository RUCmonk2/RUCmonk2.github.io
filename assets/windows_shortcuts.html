Windows & VS Code 快捷技巧速查

面向计算机专业/开发者的常用组合键与小技巧，覆盖系统、编辑器和终端。

返回博客

按场景整理。优先熟悉：Win + .（符号面板）、Win + V（剪贴板历史）、Win + Shift + S（截图）、Ctrl + Shift + P（VS Code 命令面板）、Ctrl + Shift + N（新开 VS Code 窗口）。

输入与剪贴板

Win + .
打开 emoji / 符号 / 表情面板，输入特殊符号。
Win + V
剪贴板历史；先在设置里开启，可粘贴最近复制的多条记录。
Win + ;
与 Win + . 同效果（emoji 面板）。
Win + Shift + S
调出截图工具，矩形/窗口/全屏截图，保存到剪贴板。
Win + Space
切换输入法/键盘布局。

窗口与桌面

Win + Tab
任务视图，查看所有窗口并切换虚拟桌面。
Win + Ctrl + D
新建一个虚拟桌面。
Win + Ctrl + F4
关闭当前虚拟桌面。
Win + ↑ / ↓ / ← / →
窗口贴边/分屏/最大化/还原。
Win + L
快速锁屏，离开座位首选。
Win + D
显示/隐藏桌面。
Alt + Tab
在当前打开的窗口间快速切换。

启动与系统

Win + E
打开文件资源管理器。
Win + R
运行窗口，输入 `cmd`、`powershell` 或应用路径快速启动。
Win + I
打开系统设置。
Win + P
投影/显示模式切换（复制、扩展等）。
Win + X
电源用户菜单：终端、任务管理器、设置等入口。
Ctrl + Shift + Esc
直接打开任务管理器。

通用编辑与导航

Ctrl + C / X / V
复制 / 剪切 / 粘贴。
Ctrl + Z / Y
撤销 / 重做。
Ctrl + A
全选当前区域内容。
Ctrl + F
查找；大多数应用通用。
Ctrl + H
替换（在支持的编辑器/浏览器中）。
Ctrl + L
在浏览器/资源管理器聚焦地址栏。
Alt + 左/右
在浏览器/资源管理器中后退/前进。

VS Code 高效操作

Ctrl + Shift + P
命令面板，几乎所有操作的入口。
Ctrl + P
快速打开文件；可输入 `@` 跳转符号，`:` 跳转行。
Ctrl + Shift + N
新开一个 VS Code 窗口。
Ctrl + `
内置终端开关。
Ctrl + B
隐藏/显示侧边栏。
Alt + ↑ / ↓
移动当前行/选区。
Alt + Shift + ↑ / ↓
复制当前行/选区并向上/下插入。
Ctrl + D
选择下一个相同文本进行多光标编辑。
Ctrl + /
行注释切换。

终端 / PowerShell / Windows Terminal

Ctrl + L
清屏（Windows Terminal/Pwsh 支持）。
Ctrl + Shift + C / V
终端内复制/粘贴（Windows Terminal 默认）。
Ctrl + Shift + T
在 Windows Terminal 重新打开上次关闭的标签。
Alt + Shift + D
在 Windows Terminal 分屏（纵向）。

截图与录屏

Win + Shift + S
截图（矩形/窗口/全屏），存入剪贴板。
Win + Alt + R
快捷录屏（Xbox Game Bar），适合简单录制。
Win + G
打开 Xbox Game Bar，可录屏/音量调节/性能监控。
