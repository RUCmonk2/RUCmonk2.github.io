# 常用提示词

最后更新：2026-07-09

## 给新项目初始化交接机制

```text
请参考 Desktop/codex交互指南，为 <目标项目路径> 建立 Codex 交互入口、temps 临时资料区和 codex 语境交接机制。先检查目标项目已有结构和同名文件，再告诉我你准备创建或修改哪些文件，确认后执行。
```

## 简化版

```text
请参考 Desktop/codex交互指南，给这个项目补齐 Codex 交互入口、temps 和语境交接文件。
```

## 新窗口恢复上下文

```text
请先阅读 <目标项目路径>/codex交互reademe.md 恢复上下文。先不要修改文件，先告诉我你理解到的当前状态、重要文件夹、temps 临时资料处理规则、遗留问题和下一步建议。
```

## 处理 temps 临时资料

```text
请先阅读 <目标项目路径>/codex交互reademe.md 恢复上下文，然后检查 temps/ 里的临时资料。请先告诉我你看到哪些资料、判断它们应该归类到哪里、哪些信息需要提取、哪些文件处理完可以删除。确认后再执行移动、整理或删除。
```

## 结束前更新交接文件

```text
请更新 <目标项目路径>/codex交互reademe.md 和 <目标项目路径>/codex语境交接/，记录本次完成内容、当前遗留问题、重要文件变化和下一步建议。
```

## 只让 Codex 复述需求和计划

```text
先不要修改文件。请复述我的需求，并给出你打算怎么做、会修改哪些文件、可能有哪些风险。
```

## 检查交接系统是否完整

```text
请参考 Desktop/codex交互指南，检查 <目标项目路径> 里的 Codex 交接系统是否完整，包括 codex交互reademe.md、codex语境交接/ 和 temps/。只检查并列出缺失项，先不要修改文件。
```
