# {{PROJECT_NAME}} temps 临时资料区

最后更新：{{DATE}}

这个目录用于临时放置还没分类、还没决定去向的项目资料，例如：

- 刚收到的截图、录屏、音频、压缩包；
- 临时下载的论文、网页截图、代码片段；
- 待 Codex 提取信息、转写、分类或整理的材料；
- 只为一次对话提供上下文的中间文件。

## 使用规则

1. 放进这里的内容默认不是正式材料。
2. Codex 处理前应先列出本目录文件，并说明每个文件建议如何处理。
3. 有长期价值的内容应移动到正式目录，或把信息整理进正式 Markdown / LaTeX / 代码文件。
4. 只做一次性参考的内容，处理后可以清理；无法判断能否删除时先问用户。
5. 如果某个临时材料已经被整理到正式文件，应在下面的清理记录中标注去向。

## 建议清理节奏

- 每次阶段性任务结束时，快速检查一次。
- 每周至少清理一次，建议放在周日或下次开始项目前。
- 当本目录文件超过 10 个，或总大小超过 200MB 时，优先清理。
- 新开 Codex 窗口恢复上下文后，如果用户说“看看临时资料”，先从这里开始。
